[![Python Version](https://img.shields.io/badge/python-3.8%2B-blue.svg)](https://www.python.org/)

پروژه VisionMouse-AI یک سیستم پیشرفته کنترل ماوس بدون لمس مبتنی بر بینایی ماشین است. این سیستم با استفاده از پردازش تصویر OpenCV و کتابخانه MediaPipe، حرکات دست را در زمان واقعی (Real-time) پردازش کرده و آن‌ها را به دستورات حرکتی ماوس، کلیک چپ و راست، درگ کردن و اسکرول تبدیل می‌کند.

## 🚀 ویژگی‌های کلیدی

* حرکت دقیق نشانگر ماوس: استفاده از ردیابی نقاط کلیدی دست به همراه الگوریتم نرم‌سازی برای جلوگیری از لرزش نشانگر.
* کنترل جداگانه دو دست: نگاشت حرکات مجزا برای دست راست و چپ جهت جلوگیری از تداخل دستورات.
* پشتیبانی از نمایشگرهای با دقت بالا (High DPI): ادغام شده با APIهای سیستم‌عامل ویندو (User32) برای پشتیبانی کامل از مانیتورهای کیفیت بالا (4K/1080p).
* عملیات کامل ماوس بدون لمس: پشتیبانی از کلیک چپ، کلیک راست، نگه داشتن و کشیدن (Drag) و اسکرول عمودی.

## 🖐️ نقشه حرکات دست (Gesture Mapping)

### 🖐️ دست راست (جابه‌جایی و کلیک راست)
* حرکت ماوس: نوک انگشت اشاره (Landmark 8) را در مقابل دوربین حرکت دهید.
* کلیک راست: نزدیک کردن نوک انگشت شست (Landmark 4) و نوک انگشت اشاره (Landmark 8) به یکدیگر (فاصله کمتر از 22 پیکسل).

### 🤚 دست چپ (کلیک چپ، درگ و اسکرول)
* کلیک چپ: یک چسباندن سریع (Pinch) بین نوک انگشت شست (Landmark 4) و نوک انگشت اشاره (Landmark 8).
* کشیدن و رها کردن (Drag & Drop): نگه داشتن حالت Pinch بین شست و اشاره به مدت بیشتر از 0.4 ثانیه؛ با باز کردن انگشتان، حالت Drag رها می‌شود.
* اسکرول به بالا: بالا بردن نوک انگشت وسط (Landmark 12) از حد مجاز تعیین شده.
* اسکرول به پایین: پایین آوردن نوک انگشت اشاره (Landmark 8) از حد مجاز تعیین شده.

## 🛠️ پیش‌نیازها و راهنمای نصب

### پیش‌نیازها
* سیستم‌عامل Windows (استفاده از فراخوانی‌های ctypes Win32 API)
* پایتون نسخه 3.8 یا بالاتر
* وب‌کم (داخلی یا خارجی)

۱. مخزن را کلون کرده و وارد پوشه پروژه شوید:
git clone https://github.com/YourUsername/VisionMouse-AI.git
cd VisionMouse-AI

۲. یک محیط مجازی پایتون (Virtual Environment) ایجاد و فعال کنید:
python -m venv venv
source venv/bin/activate  # در ویندوز: venv\Scripts\activate

۳. کتابخانه‌های مورد نیاز را نصب کنید:
pip install opencv-python mediapipe numpy

## 💻 نحوه اجرا

اسکریپت اصلی را از طریق ترمینال اجرا کنید:

python gesture_mouse.py

برای خروج ایمن از برنامه، دکمه 'q' را روی پنجره دوربین فشار دهید یا در ترمینال Ctrl+C بزنید.

## 🧠 معماری سیستم و نرم‌سازی حرکت (Smoothing)

این برنامه تصاویر را با استفاده از ماژول Hands در MediaPipe جهت پردازش سریع و بدون تاخیر دریافت می‌کند. برای نگاشت مختصات تصویر دوربین (640x480) به ابعاد کامل مانیتور از درایه‌ی خطی (np.interp) استفاده شده است. همچنین جهت حذف لرزش‌های دست، فاکتور میانگین متحرک نمایی (smoothening = 9) روی مختصات ماوس اعمال می‌شود.
